Alert 提示框 ​

效果预览 ​

💡
系统提示
检测到新固件版本 v2.1.0。
✅
操作成功
设备已添加到您的账户。
⚠️
注意
电池电量低于 20%。
❌
错误
设备连接断开，请检查网络。

极简代码 ​

一个提示框 = .alert + 类型 .alert-{type} + 图标 + 内容：

html
<div class="alert alert-info">
  <span class="alert-icon">💡</span>
  <div class="alert-content">
    <div class="alert-title">提示</div>
    <div>这是一条提示信息</div>
  </div>
</div>

最小版本（不要标题）：

html
<div class="alert alert-success">✅ 保存成功！</div>

修改和应用 ​

改类型 ​

class颜色用途
.alert-info蓝信息提示
.alert-success绿成功
.alert-warning黄警告
.alert-error红错误

效果：

💡 info 提示
✅ success 提示
⚠️ warning 提示
❌ error 提示

子元素 ​

class作用
.alert-icon左侧图标（emoji 或 SVG）
.alert-content内容容器
.alert-title标题（粗体）
html
<div class="alert alert-warning">
  <span class="alert-icon">⚠️</span>
  <div class="alert-content">
    <div class="alert-title">这是标题</div>
    <div>这是详细说明文字...</div>
  </div>
</div>

效果：

⚠️
这是标题
这是详细说明文字...

不要图标 ​

直接放文字也行，自动左对齐：

html
<div class="alert alert-error">连接失败，请重试。</div>

效果：

连接失败，请重试。

实际场景 ​

表单验证失败 ​

html
<form>
  <label>邮箱</label>
  <input type="email" value="invalid-email">
  <div class="alert alert-error mt-2">
    <span class="alert-icon">❌</span>
    <div class="alert-content">邮箱格式不正确</div>
  </div>
</form>

效果：

❌
邮箱格式不正确

系统通知区 ​

html
<div class="flex flex-col gap-2">
  <div class="alert alert-info">💡 系统将于今晚 22:00 维护</div>
  <div class="alert alert-success">✅ 5 台设备已成功添加</div>
</div>

效果：

💡 系统将于今晚 22:00 维护
✅ 5 台设备已成功添加

常见错误 ​

❌ 不要✅ 应该
Alert 写很长段落Alert 应保持简短，详细用 Modal
严重警告用 Alert关键警告用 Modal 更醒目
Alert 里塞很多按钮Alert 只是提示，操作应该用 Modal
